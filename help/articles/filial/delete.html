Удаление филиала

Точка входа:
  • http://188.246.224.156/api/v1/filial/delete.php
Обязательные параметры:
  • id (число) - ID удаляемого филиала

Внимание!

Скрипт принципиально не может запросить подтверждение на удаление. Поэтому, если ID указан, то будут удалены все данные, относящиеся к указанному филиалу.
Сообщения об ошибках:
  • ERROR_EMPTY_ID - не указан ID удаляемого филиала
Пример: http://188.246.224.156/api/v1/filial/delete.php?id=1
