Список админов филиала

Точка входа:
  • http://188.246.224.156/api/v1/filial/admins.php
Фильтр полей:
  • Для вывода не всех полей укажите в GET-строке параметр fields=
    • ID - ID пользователя в системе
    • username - имя пользователя
    • password - пароль
    • salt - соль для генерации пароля
    • status - статус (0 - суперхост, 1 - хост, 2 - разработчик, 3 - админ клуба, 4 - зарезервировано, 5 - обычный пользователь)
    • email - email пользователя
    • phone - номер телефона
    • name - имя
    • surname - фамилия
    • birthdate - дата рождения
    • reg_date - дата регистрации
    • last_visit - последний визит
    • phone_code - код отправленный на телефон для проверки (если ячейка пуста - телефон подтвержден)
    • email_token - токен для проверки email (если пустота - email подтвержден)
    • banned - Если 1 - пользователь забанен )))
    • ban_reason - причина похода в баню
    • ban_reason_admin - причина бана для администратора (пользователь ее не видит)
    • ban_start - когда забанен
    • ban_end - конец бана
    • game_time - сколько пользователь наиграл в минутах
    • rang - ранг пользователя (список рангов хранится в таблице tg_rang, доступ через /api/v1/rang/*.*)
    • discount - скидка, которой пользуется пользователь в процентах
    • filial_id - ID филиала, в котором зарегистрирован пользователь
    • temp - если 1, то пользователь временный
    • ref_user - ID пользователя, под которым пользователь зарегистрировался
    • ref_percent - процент, который получает пользователь как реферал
    • abonement - дата и время завершения абонемента
    • send_sms - если 0, то пользователь разрешил рассылку смс
    • send_push - если 0, то пользователь разрешил рассылку пушей

    Поля перечисляются через запятую без пробелов, например: admin.php?fields=username,email,phone

Фильтрующие параметры:
  • id (число, список через запятую) - по ID пользователя
  • filial_id (число, список через запятую) - по ID филиала(ов)
  • filial_ip (строка) - по IP-адресу клуба (если указать filial_ip=auto, то будет использован IP, с которого происходит обращение к скрипту)
  • city (строка) - по городу клуба (? заменяет один символ, * заменяет произвольное количество символов)
Для сортировки укажите параметр order=
  • Любое из полей, перечисленных в фильтре полей
Ограничение количества элементов в выборке:
  • limit={количество в виде числа}
Пропустить некоторое количество записей в начале:
  • offset={сколько пропустить в виде числа}
Формат вывода:
  • format=
    • ini (в виде ini-файла)
    • json (JSON) - по умолчанию
Все параметры являются необязательными. Будут выведены только пользователи со статусом 1 и 3 (хост и админ клуба)
Пример: http://188.246.224.156/api/v1/filial/admins.php?format=ini&fields=email,username&ip=95.181.143.212
