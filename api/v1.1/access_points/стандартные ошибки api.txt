<pre>Эти ошибки могут появиться при работе любого скрипта, требующего авторизации

ERROR_USER_NOT_AUTHORISED - пользователь не авторизован
ERROR_USER_NOT_EXISTS - пользователь с указанными данными не существует
ERROR_INVALID_PASS - пароль, сохраненный в сессии неверен (возможно, он был изменен во время работы)
ERROR_USER_BANNED - текущий пользователь забанен
ERROR_PERMISSION_DENIED - пользователь не имеет права на текущую операцию
