<pre>Точка входа: https://truegamers.pro/api/v1.1/computers
Пример GET-запроса: https://truegamers.pro/api/v1.1/computers/?filter={"branch_id":1},"sort":{"field":"ID","order":"ASC"}}
Метод: GET / POST
Авторизация: Обязательна
Параметры:
	filter=строка         //JSON-массив с ID филиала и параметрами сортировки (пример: {"branch_id":[1,3]},"sort":{"field":"ID","order":"ASC"}})
Формат параметров:
	filter={
			"branch_id":1 / [1,3] / [] //через слэш указаны допустимые форматы параметра
			"sort":{
				"field":"ID" //название поля для сортировки (список полей ниже)
				"order":"ASC" //направление сортировки (ASC / DESC)
			}
		}
Поля для сортировки:
	ID - по ID компьютера (большими буквами)
	user_id - по ID пользователя, находящегося за компьютером
	comp_id - по номеру компьютера в филиале
	category - по номеру зала
	status - по статусу (выключен/свободен/занят)
	branch_id - по ID филиала
	money - по балансу пользователя, находящегося за компьютером
	bonus - по бонусному балансу пользователя
	packet_id - по ID тарифа по которому играет пользователь
	minutes - по остатку минут у пользователя
	pos_x - по x-позиции на карте клуба
	pos_y - по y-позиции на карте клуба
	mac - по mac-адресу
	ip - по IP-адресу в сети филиала
	version - по версии клиентского ПО
	
Возможные ошибки:
	Только стандартные
